# How to Read a Photograph

An interactive classroom presentation for Year 9 English — Visual Literacy unit, based on the NYT *'What's Going On in This Picture?'* method.

## What it does

A click-through slide deck (14 slides) that guides students through a 5-step photograph analysis framework: **Observe → Question → Context → Conclude → Message**. Three real press photographs are embedded for in-class analysis and discussion, followed by a structured writing task.

## Files

| File | Description |
|------|-------------|
| `index.html` | The presentation — open this in a browser |
| `VTS11-17-14LN-superJumbo.webp` | Photo 1 — Gaza |
| `VTS02-22-16LN-superJumbo.webp` | Photo 2 — Elephant, India |
| `VTS11-03-14LN-superJumbo.webp` | Photo 3 — Children/aircraft, West Africa |

## How to use

Open `index.html` in a browser. Navigate with **arrow keys**, **spacebar**, or the **← →** buttons in the bottom-right corner. All images must be in the same folder as the HTML file.

## To swap photos

Replace any of the three image files and update the corresponding `src` attribute in `index.html`:

```html
<img src="YOUR-FILE.jpg" alt="Photo 1" ...>
<img src="YOUR-FILE.jpg" alt="Photo 2" ...>
<img src="YOUR-FILE.jpg" alt="Photo 3" ...>
```
